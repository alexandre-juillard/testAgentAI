# But du projet

Le but de la création de ce projet est de mettre en place un agent AI afin de répondre à un besoin.
Mon besoin est le suivant : 


- avoir un agent autonome croné
- l'agent va sur des sites spécialisés
- l'agent me renvoie des informations résumées
- l'agent est capable de me proposer des exercices pour approfondir mes connaissances

Ce projet n'a aucune vocation commerciale, juste un entrainement personnel à la connexion des agents IA. Il restera open-source pour qui voudra s'en servir. Aucune information sensible ne sera publiée.

## Choix technos

Le projet se fera en Nodejs, v24.14
La gestion des packages se fera avec npm, v11.9
Un backend devrait suffir, à voir avec l'usage (il s'agit d'un test).

## Package installés avec npm : 

discord.js / openai / node-cron / axios / dotenv / rss-parser

### Leurs utilités

discord.js : bot Discord

openai : IA

node-cron : tâches planifiées

axios :	appels API

dotenv : gestion clés

rss-parser : lire flux news
